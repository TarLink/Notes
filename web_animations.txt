ELEM|CSSPSEUDOELEM:
  - animate([KEYFRAMES[, KEYFRAMEANIMATIONOPTIONS]])->ANIMATION

KEYFRAMES either:
  - OBJ_ARR:
     - PROP: VAL
     - offset NUM (0-1)
     - easing STR
     - composite STR
  - OBJ:
     - PROP: VAL_ARR
  - SHAREDKEYFRAMELIST
SHAREDKEYFRAMELIST:
  - new SharedKeyframeList(KEYFRAMES)

ANIMATIONEFFECTTIMING[READONLY]:
  - delay NUM (def: 0)
  - endDelay NUM (def: 0)
  - duration NUM (def: 0)
  - direction STR: 'normal' (def), 'reverse', 'alternate[-reverse]'
  - fill STR: 'none' (def), 'forwards', 'backwards', 'both', 'auto'
  - easing STR: 'linear' (def), 'ease[-in|out|in-out]', 'cubic-bezier(4 NUM)'
  - iterationStart NUM (def: 0)
  - iterations NUM (def: 1, can be Infinity)
KEYFRAMEEFFECTOPTIONS:
  - inherits from ANIMATIONEFFECTTIMING
  - composite STR: 'replace' (def), 'add', 'accumulate'
  - iterationComposite STR: 'replace' (def), 'accumulate'
  - spacing STR: 'distribute' (def), 'paced'
KEYFRAMEANIMATIONOPTIONS
  - inherits from KEYFRAMEEFFECTOPTIONS
  - id STR
COMPUTEDTIMINGPROPERTIES:
  - inherits from ANIMATIONEFFECTTIMING
  - endTime NUM
  - activeDuration NUM
  - localTime NUM|null
  - progress null|NUM (0-1)
  - currentIteration null|NUM (starts at 0)

ANIMATIONEFFECT[READONLY]:
  - timing ANIMATIONEFFECTTIMING[READONLY]
  - getComputedTiming()->COMPUTEDTIMINGPROPERTIES
KEYFRAMEEFFECT[READONLY]: 
  - inherits from ANIMATIONEFFECT[READONLY]
  - new KeyframeEffect[ReadOnly]([ELEMENT[, KEYFRAMES[, KEYFRAMEEFFECTOPTIONS]]])
  - target ELEM|null
  - composite, iterationComposite, spacing: see KEYFRAMEEFFECTOPTIONS
  - getKeyframes()->KEYFRAMES
     - using OBJ_ARR syntax
     - also add:
        - computedOffset NUM
  - setKeyframes(KEYFRAMES): not if readonly

ELEM|CSSPSEUDOELEM|DOCUMENT:
  - getAnimations()->ANIMATION_ARR
ANIMATION:
  - new Animation([ANIMATIONEFFECTREADONLY[, ANIMATIONTIMELINE]])
     - def ANIMATIONTIMELINE: DOCUMENT.timeline
  - currentTime NUM|null
  - startTime NUM
  - effect ANIMATIONEFFECTREADONLY
  - finished PROMISE(ANIMATION)
  - ready PROMISE(ANIMATION)
  - id STR
  - playState STR: 'idle', 'pending', 'running', 'paused', 'finished'
  - playbackRate NUM (def: 1)
  - timeline ANIMATIONTIMELINE
  - oncancel = FUNC(ANIMATIONPLAYBACKEVENT)
  - onfinish = FUNC(ANIMATIONPLAYBACKEVENT)
  - cancel()
  - finish()
  - pause()
  - play()
  - reverse()
ANIMATIONPLAYBACKEVENT:
  - inherits from EVENT
  - currentTime NUM|null
  - timelineTime NUM|null

DOCUMENT:
  - timeline DOCUMENTTIMELINE
ANIMATIONTIMELINE
  - currentTime NUM|null
DOCUMENTTIMELINE:
  - inherits from ANIMATIONTIMELINE
  - new DocumentTimeline(NUM)
