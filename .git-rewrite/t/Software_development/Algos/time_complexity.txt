Time complexity :
  - temps minimal requis pour accomplir une fonction ayant un input de taille n
  - noté T(n)
Types (k étant une constante, et poly(n) tout polynome possible avec n comme seule variable) :
  - constante : O(1)
    - exemple : lire array[index] 
  - log-logarithmique : O(log(log(n)))
  - logarithmique : O(log(n))
    - ex : binary search
  - polylogarithmique : O((log(n))^k) 
  - fractional power : O(n^k) (où k est entre 0 et 1)
  - linéaire : O(n)
    - ex : max(array)
  - linéarithmique : O(n.log(n))
    - heap sort
  - subquadratric : O(n²)
    - bubble sort
  - cubic time : (n³)
  - polynomial : O(poly(n))
  - exponential : O(2^poly(n))
  - double-exponential : O(2^(2^(poly(n)))
